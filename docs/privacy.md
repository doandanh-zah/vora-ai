## 2. Information we collect

Depending on how you use VORA, we may process:

### a) Account and basic profile data
- Name, username, email (if provided)
- App/device identifiers

### b) Voice and command data
- Voice input during active sessions
- Transcribed text commands
- Follow-up command context for task completion

### c) Usage and diagnostics
- Feature usage events
- Error logs, crash logs, performance metrics
- Device/app version information

### d) Action context data
- Selected text, clipboard content, current app/window metadata
- Browser URL/title metadata (when needed for tasks)
- Screenshots only when explicitly requested or required for a task

### e) Integrations data
- Data needed to execute user-requested actions in connected services (e.g., messaging apps)

---

## 3. How we use information

We use data to:
- Provide and operate VORA features
- Understand and execute user commands
- Improve reliability, safety, and user experience
- Detect abuse, prevent fraud, and enforce platform security
- Comply with legal obligations

We do **not** sell personal data.

---

## 4. Voice and session behavior

- Wake word detection may run locally on device.
- Voice processing sessions are activated after wake/trigger.
- Audio is processed only as needed to complete user-requested tasks.
- We aim for event-driven processing (not always-on heavy processing).
