# `vora setup`

Initialize `~/.vora/vora.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
vora setup
vora setup --workspace ~/.vora/workspace
```

To run onboarding via setup:

```bash
vora setup --wizard
```
